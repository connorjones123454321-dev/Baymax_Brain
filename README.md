# Baymax
This is a project to create baymax

### Requirements
the pyttsx3 package requires python 3.11 or lower
` sudo apt install espeak espeak-ng espeak-ng-data `

## Setup 
install requirements
``` bash
sudo apt install portaudio19-dev python3-dev
python3.11 -m pip install -r requirements.txt
```

## How to Run
``` bash
source baymax_env/bin/activate
```

``` bash
python3.11 baymax_safe.py
```
